# 🚀 Crypto Tracker

A modern cryptocurrency tracking web application built using React.js and Vite. This project provides real-time cryptocurrency market data, price trends, and detailed coin information through a clean and responsive user interface.

---

## 📌 Features

- 📈 Real-time cryptocurrency price tracking
- 🔍 Search and explore cryptocurrencies
- 📊 Interactive charts and market trends
- 📱 Fully responsive design
- ⚡ Fast and optimized performance with Vite
- 🌐 API integration for live crypto data
- 🪙 Detailed coin information pages

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Vyankatesh644/crypto-tracker.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd crypto-tracker
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

---

## 🌐 Run Locally

After running the development server, open:

```bash
http://localhost:5173
```
